========================ETAP I========================================================================================================================================================================
Projekt składa się z 2 klas, klasy automatu(machine.cpp) oraz klasy napoju(drink.cpp)

Klasa automat odpowiada za przechowywanie i sprzezdaż obiektów klasy drink a także za tworzenie interfejsu użytkownika.

Automat do momentu zamówienia błędnej pozycji lub do wyjścia z niego po udanym zakupie

Program wyswietla na ekranie naszkicowany automat, wraz z możliwymi do zakupu napojami, puste miejsca w automacie są oznaczone napisem EMPTY SLOT

Automat może przechowywać 16 produktów na każdej stronie strony można zmieniać wpisując w momencie wyboru ID -1 lub -2

Za drukowanie automatu odpowiada funkcja print void print( drink *stock, int size, std::string id="-"), a także przeładowana funkcja prywatna print - która służy do drukowania niestandarwoych elementów automatu (jakich jam klawiatura oraz wybrane ID),
w wyświetlaniu automatu przydatny jest również przełądowany operator << klasy drink, dzięki któremu w sposób czytelny wypisujemy dane.

===Opis funkcji print===

-Najbardziej zagmatwana funkcja w projekcie, odpowiada za wypisywanie produktów i rysowanie automatu,

-jako argumenty przyjmuje tablice z napojami, jej rozmiar oraz string o nazwie id

-W przypadku, kiedy nie jest zamówiony żaden napój, string ID przyjmuje domyślną wartość "-",

-w przypadku kiedy napój jest zamówiony wartość ta ulega zmianie na numer indeksu pod którym znajduje się zamówiony napój (ponieważ id jest numerem konwertujemy je do stringa) //prykład wywołania w funckji change()

-std::cout << std::fixed; używamy aby zawsze 2 miejsca po precinku wypisywać w cenie nawet jeśli po przecinku jest 3.4 wypiszemy 3.40 ( bez fixed liczyłyby się dwa miejsca nie po precinku ale w ogóle.)

-int pos = 16 * ( page - 1 ); dzięki tej linii tworze kolejne strony, numer strony jest zmiennąklasy machine domyślnie ustawioną na jeden a więc 1 strona wyświetli napoje na indeskach od 0 do 15 druga od 16 do 31 itd...

- std::string hash = ( i >= KEYPAD_START && i < KEYPAD_STOP ) ? print( index, id ): "|###################|"; - jeśli znajdujemy się na linii o indeksie nieregularnym(takiej linii na której powinno pojawić domyślne |###################| wywołujemy prełąwodanąfunckje print )

- preładowana funckja print odpowiada za zwracanie stringa z nieregularną linią, a także za umieszczanie id zamówionego produktu w menu wyboru;
